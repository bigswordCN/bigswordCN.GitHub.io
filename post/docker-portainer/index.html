安装Docker可视化图形工具Portainer

2022-06-11

安装portainer

docker run -d -p 8000:8000 -p 9000:9000 --name portainer \
    --restart=always \
    -v /var/run/docker.sock:/var/run/docker.sock \
    -v portainer_data:/data \
    portainer/portainer-ce:latest

-v参数表示创建映射目录，位于默认映射位置，方便重新拉取镜像后升级。
